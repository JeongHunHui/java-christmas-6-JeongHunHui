# 🎄12월 이벤트 플래너🎄

고객이 방문할 날짜와 메뉴를 미리 선택하면 이벤트 플래너가 아래 내용을 보여준다.

- 주문 메뉴
- 할인 전 총주문 금액
- 증정메뉴
- 혜택 내역
- 혜택 금액
- 할인 후 예상 결제 금액
- 12월 이벤트 배지 내용

### 메뉴

```
<애피타이저>
양송이수프(6,000), 타파스(5,500), 시저샐러드(8,000)

<메인>
티본스테이크(55,000), 바비큐립(54,000), 해산물파스타(35,000), 크리스마스파스타(25,000)

<디저트>
초코케이크(15,000), 아이스크림(5,000)

<음료>
제로콜라(3,000), 레드와인(60,000), 샴페인(25,000)
```

### 달력

- 전체 이벤트 기간은 2023년 12월 1일부터 12월 31일 까지
- 매주 금요일, 토요일이 주말, 나머지는 평일
- 3, 10, 17, 24, 25, 31일은 특별 할인을 적용

### 12월 이벤트 계획

- 크리스마스 디데이 할인
    - 이벤트 기간: 2023.12.1 ~ 2023.12.25
    - 적용 대상: 모든 고객
    - 적용 혜택: 1,000원으로 시작하여 크리스마스가 다가올수록 날마다 할인 금액이 100원씩 증가하고, 총주문 금액에서 해당 금액만큼 할인 (e.g. 시작일인 12월
      1일에 1,000원, 2일에 1,100원, ..., 25일엔 3,400원 할인)
- 평일 할인
    - 이벤트 기간: 2023.12.1 ~ 2023.12.31
    - 적용 대상: 일요일~목요일에 방문한 고객
    - 적용 혜택: 평일에는 디저트 메뉴를 메뉴 1개당 2,023원 할인
- 주말 할인
    - 이벤트 기간: 2023.12.1 ~ 2023.12.31
    - 적용 대상: 금요일, 토요일에 방문한 고객
    - 적용 혜택: 메인 메뉴를 메뉴 1개당 2,023원 할인
- 특별 할인
    - 이벤트 기간: 2023.12.1 ~ 2023.12.31
    - 적용 대상: 이벤트 달력에 별이 있는 날에 방문한 고객
    - 적용 혜택: 총주문 금액에서 1,000원 할인
- 증정 이벤트
    - 이벤트 기간: 2023.12.1 ~ 2023.12.31
    - 적용 대상: 할인 전 총주문 금액이 12만 원 이상
    - 적용 혜택: 샴페인 1개 증정

# ⏳ 어플리케이션 흐름

### 1. 방문 날짜 입력

- 방문 날짜 입력 메시지를 출력한다.
- 방문 날짜를 입력받는다.

### 2. 주문할 메뉴와 개수 입력

- 주문할 메뉴와 개수 입력 메시지를 출력한다.
- 주문할 메뉴와 개수를 입력받는다.

### 3. 주문 정보 및 혜택 정보 출력

- 방문 날짜와 주문할 메뉴를 통해 받게될 혜택 정보를 계산해 출력한다.

---

# ✏️ 구현할 기능 목록

### 공통

- [x]  잘못된 입력으로 예외 발생 시 에러 메시지 출력 후 다시 입력 받는다.
- [x]  모든 에러 메시지는 `[ERROR]`로 시작한다.

### 방문 날짜 입력

- [x]  입력 받기 전에 아래의 메시지를 출력한다.

  ```
  12월 중 식당 예상 방문 날짜는 언제인가요? (숫자만 입력해 주세요!)
  ```

- [x]  방문 날짜를 입력 받는다.

  ```
  -유효하지 않은 날짜 입력시 에러 메시지-
  [ERROR] 유효하지 않은 날짜입니다. 다시 입력해 주세요.
  ```

- [x]  입력 받은 방문 날짜가 1이상 31이하가 아닌 경우 예외를 발생시킨다.

### 메뉴 주문

- [x]  주문 정보 입력 메시지를 출력한다.

  ```
  주문하실 메뉴와 개수를 알려 주세요. (e.g. 해산물파스타-2,레드와인-1,초코케이크-1)
  ```

- [x]  주문할 메뉴와 개수를 입력받는다.

  ```
  메뉴 이름과 개수는 {메뉴_이름}-{메뉴_개수}와 같은 형식으로 입력한다.
  ','로 구분하여 여러개 입력할 수 있다.
  
  -예시 입력-
  티본스테이크-1,바비큐립-1,초코케이크-2,제로콜라-1
  ```

  ```
  -유효하지 않은 주문 입력시 에러 메시지-
  [ERROR] 유효하지 않은 주문입니다. 다시 입력해 주세요.
  ```

- [x]  메뉴 이름과 개수가 `{메뉴_이름}-{메뉴_개수},{메뉴_이름}-{메뉴_개수}...`와 같은 형식이 아니면 예외를 발생시킨다.
- [x]  메뉴 이름이 메뉴판에 있는 메뉴의 이름이 아니면 예외를 발생시킨다.
- [x]  메뉴 개수는 1 이상의 정수가 아니면 예외를 발생시킨다.
- [x]  주문할 메뉴가 중복되면 예외를 발생시킨다.

  ```
  티본스테이크-1,티본스테이크-2 // 유효하지 않은 주문
  ```

- [x]  음료만 주문 시 예외를 발생시킨다.
- [x]  메뉴 개수의 총합이 20을 넘기면 예외를 발생시킨다.

  ```
  시저샐러드-1,티본스테이크-1,크리스마스파스타-1,제로콜라-3,아이스크림-1
  -> 위와 같은 입력 시 총개수는 7개
  ```

### 적용 혜택 계산

- [x]  이벤트 기간에 방문했는지 확인한다.
- [x]  할인 전 총주문 금액을 계산한다. (주문한 모든 메뉴들의 `메뉴 가격 * 개수`의 총합)
- [x]  총주문 금액 10,000원 이상부터 이벤트가 적용된다. (총주문 금액이 할인 전인지 이후인지 명확하지 않음 → 할인 전 총주문 금액으로 계산)
- [x]  크리스마스 디데이 할인에 적용되는지(이벤트 기간에 방문했는지) 확인한다.
- [x]  크리스마스 디데이 할인이 적용되면 혜택을 계산한다.
- [x]  평일 할인에 적용되는지 확인한다.
- [x]  평일 할인이 적용되면 혜택을 계산한다.
- [x]  주말 할인에 적용되는지 확인한다.
- [x]  주말 할인이 적용되면 혜택을 계산한다.
- [x]  특별 할인에 적용되는지 확인한다.
- [x]  특별 할인이 적용되면 혜택을 계산한다.
- [x]  증정 이벤트가 적용되는지(총주문 금액이 120000이상인지) 확인한다.
- [x]  증정 이벤트가 적용되면 혜택을 계산한다.

### 주문 정보 및 혜택 정보 출력

- [x]  혜택 미리 보기 메시지를 출력한다.

  ```
  12월 26일에 우테코 식당에서 받을 이벤트 혜택 미리 보기!
  ```

- [x]  주문 메뉴를 출력한다. (주문 메뉴의 출력 순서는 자유)

  ```
  <주문 메뉴>
  티본스테이크 1개
  바비큐립 1개
  초코케이크 2개
  제로콜라 1개
  ```

- [x]  할인 전 총주문 금액을 출력한다.

  ```
  <할인 전 총주문 금액>
  142,000원
  ```

- [x]  증정 메뉴를 출력한다. (증정 이벤트에 해당하지 않는 경우 `없음`으로 출력)

  ```
  <증정 메뉴>
  샴페인 1개
  ```

- [x]  혜택 내역을 출력한다. (혜택 내역의 출력 순서는 자유, 적용된 이벤트가 하나도 없다면 `없음`으로 출력)

  ```
  <혜택 내역>
  크리스마스 디데이 할인: -1,200원
  평일 할인: -4,046원
  특별 할인: -1,000원
  증정 이벤트: -25,000원
  ```

- [x]  총혜택 금액을 계산한다. (혜택으로 얻은 금액을 전부 더한 금액, 증정 이벤트로 받은 메뉴의 가격도 포함)
- [x]  총혜택 금액을 출력한다.

  ```
  <총혜택 금액>
  -31,246원
  ```

- [x]  할인 후 예상 결제 금액을 계산한다. (할인 전 총주문 금액에서 총혜택 금액을 뺀 뒤 증정 이벤트로 받은 혜택 금액을 더한 금액)
- [x]  할인 후 예상 결제 금액을 출력한다.

  ```
  <할인 후 예상 결제 금액>
  135,754원
  ```

- [x]  이벤트 배지여부를 계산한다.

  ```
  총혜택 금액에 따라 아래의 이벤트 배지가 부여된다.
  - 5천원 이상: 별
  - 1만원 이상: 트리
  - 2만원 이상: 산타
  ```

- [x]  이벤트 배지을 출력한다. (이벤트 배지가 부여되지 않는 경우 `없음`으로 출력)

  ```
  <12월 이벤트 배지>
  산타
  ```

# ➕ 추가 고려 사항

### 고객에게 이벤트 주의 사항 안내

- 총주문 금액 10,000원 이상부터 이벤트가 적용됩니다.
- 음료만 주문 시, 주문할 수 없습니다.
- 메뉴는 한 번에 최대 20개까지만 주문할 수 있습니다.(e.g. 시저샐러드-1, 티본스테이크-1, 크리스마스파스타-1, 제로콜라-3, 아이스크림-1의 총개수는 7개)

→ 위 주의 사항을 따로 안내하는 기능을 추가할지 고려
